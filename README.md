# React

## 1. 개발환경 설정

- Node.js 설치
- VS code 설치
- React 프로젝트 생성
   - 원하는 위치에 빈폴더를 하나 생성
   - 해당 위치에서 PowerShell을 실행
   - npx create-react-app 'project name'
   
위의 작업을 모두 마치고 VS code에서 Open folder로 해당 프로젝트 파일을 열어주면
React 개발의 준비는 끝이난다.


## 2. 프로그램 실행

- code를 실행하기 위해 터미널을 생성
> npm start

- code를 종료하기 위해 터미널에서 사용하는 단축키
> ctrl+c -> y 혹은 ctrl+c
